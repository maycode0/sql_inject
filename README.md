# SQL 注入学习资料导航

## 1. 文档总览

当前 `docs` 目录中的学习资料如下：

- `docs/sql_injection_handbook.md`：完整学习手册，适合系统学习
- `docs/sql_injection_learning.md`：基础学习内容
- `docs/sql_injection_study_plan.md`：初级 / 中级 / 高级三阶段学习计划
- `docs/sql_injection_demo.md`：教学演示说明文档
- `docs/sql_injection_demo.py`：可直接运行的 Python 演示代码
- `docs/sql_injection_exercises.md`：练习题与答案解析
- `docs/sql_injection_mindmap.md`：脑图版总结

---

## 2. 推荐阅读顺序

如果你是第一次系统学习，建议按下面顺序阅读：

1. `docs/sql_injection_handbook.md`
2. `docs/sql_injection_demo.py`
3. `docs/sql_injection_exercises.md`
4. `docs/sql_injection_mindmap.md`

如果你希望分模块学习，也可以按下面顺序：

1. `docs/sql_injection_learning.md`
2. `docs/sql_injection_study_plan.md`
3. `docs/sql_injection_demo.md`
4. `docs/sql_injection_demo.py`
5. `docs/sql_injection_exercises.md`
6. `docs/sql_injection_mindmap.md`

## 3. 各文档适用场景

### 想快速系统入门

优先看：

- `docs/sql_injection_handbook.md`

### 想先看基础理论

优先看：

- `docs/sql_injection_learning.md`
- `docs/sql_injection_study_plan.md`

### 想看代码演示

优先看：

- `docs/sql_injection_demo.md`
- `docs/sql_injection_demo.py`

运行方式：

```bash
python docs/sql_injection_demo.py
```

如果 `python` 命令不可用：

```bash
py docs/sql_injection_demo.py
```

### 想做自测

优先看：

- `docs/sql_injection_exercises.md`

### 想快速复习

优先看：

- `docs/sql_injection_mindmap.md`

---

## 4. 推荐学习路径

### 路线 A：完整学习路线

1. 先阅读 `docs/sql_injection_handbook.md`
2. 再运行 `docs/sql_injection_demo.py`
3. 然后完成 `docs/sql_injection_exercises.md`
4. 最后使用 `docs/sql_injection_mindmap.md` 做复习

### 路线 B：分阶段学习路线

1. 用 `docs/sql_injection_study_plan.md` 确定阶段目标
2. 用 `docs/sql_injection_learning.md` 学基础内容
3. 用 `docs/sql_injection_demo.py` 做代码观察
4. 用 `docs/sql_injection_exercises.md` 做阶段检测
5. 用 `docs/sql_injection_mindmap.md` 做总结回顾
