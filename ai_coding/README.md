# ai_coding

这个目录用于存放仓库中的可运行代码示例，便于将文档资料和演示代码分开管理。

## 当前文件

- `ai_coding/sql_injection_demo.py`：SQL 注入教学演示代码，包含字符串拼接查询、参数化查询，以及排序字段白名单示例。

## 运行方式

```bash
python ai_coding/sql_injection_demo.py
```

如果本机使用的是 Windows Python Launcher：

```bash
py ai_coding/sql_injection_demo.py
```

## 使用建议

- 先阅读 `docs/sql_injection_demo.md` 了解演示目的
- 再运行 `ai_coding/sql_injection_demo.py` 观察输出差异
- 结合 `docs/sql_injection_handbook.md` 一起理解防御思路
